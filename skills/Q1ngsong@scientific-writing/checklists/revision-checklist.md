# Revision Checklist

## Structure

- [ ] The paper has one clear main claim.
- [ ] The introduction explains known, unknown, approach, and contribution.
- [ ] Each results subsection answers one question.
- [ ] Discussion starts with the answer, not a literature tour.
- [ ] Limitations are specific and proportional.

## Paragraphs

- [ ] Each paragraph has one main idea.
- [ ] Topic sentences reveal the paragraph's job.
- [ ] Similar ideas are grouped.
- [ ] Transitions signal contrast, cause, or sequence.

## Sentences

- [ ] Main subject and verb are close.
- [ ] Dead-weight openers are removed.
- [ ] Verbs are not buried in nouns.
- [ ] Long sentences are split when they carry multiple claims.
- [ ] Pronouns have clear referents.

## Evidence

- [ ] Claims match the data shown.
- [ ] Numbers in abstract, text, figures, and tables agree.
- [ ] References are primary where possible.
- [ ] Figure legends can be understood without hunting through the text.

## Submission

- [ ] Title and abstract contain reviewer-assignment keywords.
- [ ] Formatting and anonymity rules are checked.
- [ ] Supplementary material does not hide essential evidence.
- [ ] All abbreviations are introduced before use.

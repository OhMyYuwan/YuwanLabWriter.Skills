# Scientific Writing

Helps improve scientific manuscripts by diagnosing organization first, then
revising paragraphs, sentences, grammar, and submission-facing surfaces such as
titles, abstracts, introductions, figure legends, and conclusions.

Use this Skill for scientific writing, clarity revision, non-native English
support, section restructuring, grant-style text, and readability or logic
reviews. It includes a writing playbook, sentence patterns, and a revision
checklist.

This marketplace package is instruction-only. It provides writing guidance and
checklists; it does not execute code or edit documents automatically.

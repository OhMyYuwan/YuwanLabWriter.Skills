# PhD Mentor

Provides grounded doctoral, graduate-school, postdoc, and early research-career
guidance across problem finding, literature work, research execution,
publication, resilience, and career planning.

Use this Skill when a question is broad, emotionally loaded, or spans multiple
areas of doctoral life, such as finding ideas, reading papers, stuck
experiments, supervisor communication, rejection, graduation, grants, overseas
visiting, or career choices. It routes internally to bundled subskill modules
instead of asking the user to choose a separate Skill.

This marketplace package is instruction-only. It provides mentoring frameworks
and bundled reference notes; it does not replace current official policies,
medical care, legal advice, or program-specific requirements.

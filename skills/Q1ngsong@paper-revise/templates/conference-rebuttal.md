# Conference Rebuttal Template

Use when the venue allows a short PDF or text rebuttal.

## Structure

```text
We thank the reviewers for their constructive feedback. We first address the main shared concerns, then respond to reviewer-specific points.

Main concern 1: [novelty / experiment / clarity]
[Short answer + evidence + what will be clarified in final version if allowed.]

Main concern 2: [...]

R1: [specific response]
R2: [specific response]
R3: [specific response]
```

## Space Rules

- Use most space on borderline or negative concerns.
- Do not repeat the paper's abstract.
- Do not introduce new claims that require large unexplained evidence.
- If new experiments are not allowed, state policy-aware commitments carefully.

## Compact Response Pattern

```text
R2 asks whether X is fair. We clarify that [answer]. The comparison uses [same backbone/data/metric], and the result is [number/table]. We will add this clarification to Sec. [X].
```

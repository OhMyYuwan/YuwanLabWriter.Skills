# Paper Revise

Helps authors turn reviewer comments, meta-reviews, and editorial decisions
into a revision strategy, point-by-point responses, and a manuscript change
plan.

Use this Skill for rebuttals, journal response letters, major/minor revisions,
reject-and-resubmit plans, AC/AE messages, and reviews of an existing response
draft. It includes bundled templates for response letters, conference
rebuttals, AC messages, and revision checklists.

This marketplace package is instruction-only. It provides guidance, templates,
and reference notes; it does not execute code or mutate manuscripts
automatically.
